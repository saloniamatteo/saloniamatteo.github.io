TP-XFan (ThinkPad XForms Fan)

Program used to modify the fan speed of any ThinkPad. It has a graphical user interface (created with XForms arrow_forward), and the Automatic Fan Speed Management mode (AFSM).

arrow_forward
Csnake

A Snake clone that has a terminal user interface made with ncurses arrow_forward. You can save the game scores with a custom name, in any file; also, you can modify the speed/difficulty, and you can show the screen borders.

arrow_forward
Quiz

Program used to create small Quizzes and/or take a Quiz. It does not have a graphical user interface, nor does it have a terminal user interface; it only has a text-based interface. You can save the game scores with a custom name, in any file; also, it contains a simple-to-use quiz creator.

arrow_forward
Calc

A simple calculator, with support for bit-shifting, Reverse Polish Notation (RPN), and other special values. It has a text-based interface, like Quiz.
